Haven

Haven is an open-source, centralized secrets manager I built with others while transitioning from embedded systems into web development. It protects application secrets using encryption, access control, and runtime injection.

Built with Node.js and AWS, it was designed to be easy to set up and integrate into Node applications. It includes both a web UI and a CLI to make secret management less painful for developers, and it injects secrets at runtime while redacting them from stdout and stderr logs.
